# data-warehouse-project
Building modern data warehouse with Snowflake, Including ETL Processes, data modeling.

Welcome to the **Data Warehouse Project** repository. 
This projects demonstrates a comprehensive data warehousing, from building a data warehouse that support generating actionable insights. 

**designed as portfolio project.**

## Project Requirments

## Objective
Develop a modren data warehouse using Snowflake to consolidate sales data, enabling analytical reporting and informed decision-making.

## Specifications
- **Data Sources**: Import data from two source systems (ERP and CRM) provided as CSV files then loaded into S3 bucket.
- **Data Quality**: Cleanse and resolve data quality issues prior to analytics
- **Integration**: Combine both sources into a single, user-friendly data model designed for analytical queries.
- **Scope**: Historization is a must.
- **Documentation**: Provide clear documentation of the data model to support both business stakeholders and analytics teams.
